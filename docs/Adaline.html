Funkcja progowa

$ \Phi(f(z)) = \left\{ \begin{array}{ll} 1 & \ jesli\ f(z)\ge\theta\\ -1 & \ jesli\ f(z) < \theta \end{array} \right.\\ f(z)\ -\ calkowite\ pobudzenie\ po\ zastosowaniu\ funkcji\ aktywacji $

Funkcja aktywacji - Adaline implementuje liniową jej postać

$ f(z) = z $

Funkcja całkowitego pobudzenia

$ z = w_1x_1 + w_2x_2 + ... + w_mx_m $

Wartość progowa $\theta$ może zostać przeniesiona na lewą stronę równania (funkcja progowa), następnie definiując $w_0 = -\theta$ oraz $x_0 = 1$, całkowite pobudzenie przybierze postać:
$ z = w_0x_0 + w_1x_1 + w_2x_2 + ... + w_mx_m $ - jest to równanie zgodne z powyższym schematem.

Należy pamiętać jednak o dodatkowym założeniu dla funkcji progowej:

$ \Phi(f(z)) = \left\{ \begin{array}{ll} 1 & \ jesli\ f(z)\ge0\\ -1 & \ jesli\ f(z) < 0 \end{array} \right.\\ $
Waga $w_0$ jest okreslana obciążeniem jednostkowym.

Funkcja kosztu


Trening algorytmu Adaline opiera się o optymalizację funkcji kosztu. Omawiany przypadek implementuje sumę kwadratów błędów (SSE).

$ J(w) = \sum_i(y^{(i)} - f(z^{(i)}))^{2} $

Pochodna cząstkowa funkcji kosztu


$ \frac{\partial J}{\partial w_j} = \frac{\partial J}{\partial w_j} \sum_i(y^{(i)} - f(z^{(i)}))^{2}\\ \frac{\partial J}{\partial w_j} = \sum_i 2(y^{(i)} - f(z^{(i)}))\frac{\partial J}{\partial w_j}(y^{(i)} - \sum_k w_{k}x_{k}^{(i)})\\ \frac{\partial J}{\partial w_j} = 2\sum_i (y^{(i)} - f(z^{(i)}))(-x_j^{(i)}) $

Metoda gradientu prostego

Wskazana procedura pozwala iteracyjnie dojść do argumentu, dla którego optymalizowana funkcja osiąga minumum.
Poszczególne kroki optymalizacji funkcji $f(x) = x^2$:

  1. Wyznaczenie gradientu funkcji dla argumentu $x_0$ - gradientem ($\triangledown f(x)$) jest wartość pierwszej pochodnej dla wskazanego argumentu.
  2. Zmiana wartości argumentu wg. wzoru $x_0 = x_0 - n * \triangledown f(x_0)$, gdzie $n$ jest współczynnikiem uczenia.
  3. Należy powtarzać 1 oraz 2 krok aż do momentu spełniania kryterium zatrzymania

Podsumowanie

Proces "uczenia się" adaptacyjnego neuronu liniowego opiera się o optymalizację funkcji kosztu, której wartość reprezentuje błędnie sklasyfikowane obserwacje.

class Adaline():
    def __init__(self, lr = 0.05, epochs = 20, random_state = 2):
        self.lr = lr
        self.epochs = epochs
        self.random_state = random_state
    
    def fit(self, X, y):
        rand = np.random.RandomState(self.random_state)
        self.w = rand.normal(size = 1 + X.shape[1])
        self.cost = []
        for _ in range(self.epochs):
            output = self.forward(X)
            errors = (y - output)
            self.w[1:] += self.lr * X.T.dot(errors) * 2
            self.w[0] += self.lr * errors.sum() * 2
            self.cost.append((errors**2).sum())
     
    def forward(self, X):
        return np.dot(X, self.w[1:]) + self.w[0]
    
    def predict(self, X):
        return np.where(self.forward(X) >= 0.0, 1, -1)
